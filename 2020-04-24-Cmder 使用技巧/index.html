cmder 软件使用技巧

发表于 2020-04-24 更新于 2020-04-25 分类于 skill Disqus：

Windows 上的 cmder 是个很好的 terminal 工具，极力推荐，有些配置稍改下可更加得心应手，稍作纪录

Cmder 乱码

Settings -> Startup -> Environment 增加一句 set LANG=zh_CN.UTF-8

修改Tab的默认启动路径

Settings -> Stratup -> Tasks 增加 Startup dir 以及 /f 选项

Cmder 很慢的一些解决方案

  • Cmder 每次启动 Tab 都会进行一些初始化，可以对 cmder\vendor\init.bat 做如下修改，打开 log 选项和计时器，看看到底哪些步骤耗时。

  • cmder Tab 启动慢一般是 git 版本太多造成， cmder 本身已经附带了 git-for-windows，把自己装的其他版本 git 删掉，把 cmder\vendor\git-for-windows\bin 加到系统环境变量里作为默认 git 就好。
  • 根据这条post，把 cmder\vendor\clik.lua 的 git 相关检查注释掉

  • 有时候编辑完 git 信息 tab 可能会卡死，打开 任务管理器，找到 cmder 的进程名 ConEmu Emulator ，再展开，杀掉它的 git-for-windows，此时 tab 的 git 会自动退出并恢复。

修改 alias

参照格式修改 cmder\config\user_aliases.cmd 文件。
